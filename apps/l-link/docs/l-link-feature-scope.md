# L-Link 初期ローンチ機能スコープ

L-LinkはLINE外部ツールSaaSです。GARAGE LINKは在庫管理SaaS、L-touringはL-Linkを使ったバイク・車屋向けLINE構築運用代行サービスとして分離します。

## 初期スコープから除外

以下の商品・決済系は初期ローンチでは実装しません。

- 商品管理
- 販売ページ
- Stripe決済連携
- 注文管理
- 決済履歴
- 商品購入フォーム
- ASP/アフィリエイト管理

DBでも `ll_products`、`ll_orders`、`ll_payments`、`ll_product_pages`、`ll_affiliates` は作成しません。将来の販売機能として別フェーズで検討します。

## 初期必須機能

- LINE公式アカウント接続
- Webhook受信
- 友だち管理
- 友だち情報管理
- LINEプロフィール取得
- 顧客情報管理
- 個別メモ
- タグ管理
- 流入経路分析
- 回答フォーム
- リッチメニュー管理
- 一斉配信
- セグメント配信
- 予約配信
- ステップ配信
- シナリオ分岐
- 配信数管理
- スタッフ管理
- 権限管理

## LINEから自動取得できる情報

LINEプロフィールAPIで取得できる情報は以下です。

- LINE表示名
- プロフィール画像URL
- ステータスメッセージ
- 言語

友だち追加、ブロック、メッセージ、ポストバックなどのイベント日時はWebhookから管理します。

## LINEから自動取得できない情報

以下はLINEから自動取得できません。回答フォーム、手入力、またはスタッフ対応で更新します。

- 氏名
- フリガナ
- 電話番号
- メールアドレス
- 生年月日
- 性別
- 郵便番号
- 住所
- 所有車両
- 車検満了日
- 購入検討車種
- 問い合わせ種別
- 来店希望日
- 前回対応内容
- 次回対応予定日
- メモ要約

## L-touringで使う友だち情報項目

バイク・車屋向け運用では、以下を優先して管理します。

- 顧客ステータス: 見込み、商談中、既存顧客、休眠、対応不要
- 流入経路: 広告、QR、紹介、店頭、イベント、フォーム
- 興味カテゴリ: 車両購入、車検、整備、カスタム、買取、保険
- 問い合わせ種別: 購入相談、車検相談、修理相談、査定依頼、来店予約
- 所有車両
- 車検満了日
- 購入検討車種
- 希望連絡方法
- 担当スタッフ
- 次回対応予定日

車検満了日、所有車両、購入検討車種、問い合わせ種別は、L-touringの追客・リマインド・セグメント配信で使います。

## 個人情報の扱い

氏名、電話番号、メール、住所、フォーム回答、問い合わせ本文は個人情報です。監査ログやsecurityログには本文や個人情報を保存せず、業務テーブルでtenant/company単位に分離して管理します。
