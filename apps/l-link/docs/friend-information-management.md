# 友だち情報管理

L-Linkでは、LINE上の友だち情報と、店舗が管理する顧客情報を分けて扱います。

## LINEから取得する情報

`ll_line_friends` に保存します。

- LINE userId
- LINE表示名
- プロフィール画像URL
- ステータスメッセージ
- 言語
- 友だち状態
- 友だち追加日
- ブロック/解除日時
- 最終メッセージ日時
- 最終接触日時
- プロフィール取得日時
- プロフィール取得エラー

## L-Link側で管理する情報

`ll_friend_profiles` に保存します。

- 氏名、フリガナ
- 電話番号、メール、住所
- 顧客ステータス
- 流入経路
- 担当スタッフ
- 希望連絡方法
- 興味カテゴリ
- 問い合わせ種別
- 所有車両
- 車検満了日
- 購入検討車種
- 来店希望日
- 前回対応内容
- 次回対応予定日
- メモ要約

これらはLINEから自動取得できないため、回答フォームまたは手入力で更新します。

## バイク・車屋向けの用途

- 車検満了日: 車検リマインド、入庫促進
- 所有車両: 整備・点検案内、買い替え提案
- 購入検討車種: 入庫車両案内、商談フォロー
- 問い合わせ種別: 担当者振り分け、セグメント配信

## 注意事項

- TokenやSecretは画面、ログ、エラーに出しません
- LINE userIdは業務上必要な画面に限定して表示します
- フォーム回答や問い合わせ本文は監査ログ/securityログに保存しません
- 削除・退会時の完全削除/マスク処理は次フェーズで運用ルール化します
